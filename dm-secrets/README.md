# dm-secrets/ — DM EYES ONLY

> **Player:** by agreement, you don't open this folder. It holds what your character
> doesn't yet know, so the world stays full of real discovery. Peeking spoils it.

> **DM:** read and use everything here freely to run a coherent world. Rules:
> - **Never reveal** these contents except through legitimate in-world discovery.
> - When the player **earns** a piece of this knowledge, **move it** to the matching
>   public file (`world/world-bible.md`, `world/locations.md`, `world/npcs.md`,
>   `world/factions.md`) and note the in-game date of discovery.
> - Don't hint at file contents to be helpful, and don't let the world-simulator surface
>   hidden developments except as rumor, evidence, or consequence the player can chase.

## Files
- `empire-metaplot.md` — the slow-burn central plot: what the Empire is really doing.
- `hidden-lore.md` — secret history and truths of the frontier.
- `hidden-locations.md` — places not yet discovered.
- `hidden-npcs.md` — figures and their true agendas, unknown to the player.
